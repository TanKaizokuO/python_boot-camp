# Module 00 - Course Introduction

Welcome to the Free Python Course.

## What This Module Covers

- What Python is and where it is used
- How to run Python code from terminal and notebook
- Your first Python program and basic syntax

## How to Use This Course

1. Complete `getting_started.ipynb` first.
2. Move to `1.basics.ipynb` and continue in order.
3. Use `SYLLABUS.md` to track module progression.
4. Practice by running scripts in project folders after each phase.

## Run This Module

From the project root:

```bash
jupyter notebook 00_intro/getting_started.ipynb
```

## Expected Outcome

You will be able to run Python code in a notebook and understand the core syntax needed for Module 01.
