# Asynchronous Python Module

## Topic

Advanced concurrency examples with async workflows and process/thread comparisons.

## What You Will Learn

- Core `async` and `await` patterns
- Background workers and daemon behavior
- Race-condition and deadlock concepts

## What You Will Build

- Async task execution demos
- Concurrency comparison examples
- Worker and daemon experiments

## How to Run

From the project root:

```bash
python Ascynchronous_Python/13_async_python/01_async_one.py
python Ascynchronous_Python/13_async_python/02_async_two.py
python Ascynchronous_Python/13_async_python/03_async_three.py
python Ascynchronous_Python/13_async_python/09_race_condition.py
python Ascynchronous_Python/13_async_python/10_deadlock.py
```

## Expected Output

- Console logs showing async scheduling and task completion order
- Demonstrations of concurrency pitfalls and behaviors
