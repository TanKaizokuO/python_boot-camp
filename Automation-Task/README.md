# Automation Task Module

## Topic

Automation scripts for common file and system management operations.

## What You Will Learn

- Working with files and folders programmatically
- Batch operations on filenames
- Monitoring system resources from Python

## What You Will Build

- Batch rename utility
- File organizer and sorter scripts
- System resource monitor

## How to Run

From the project root:

```bash
python Automation-Task/Auto-Batch-Rename.py
python Automation-Task/Auto-File-Organizer.py
python Automation-Task/Auto-File-Sorter.py
python Automation-Task/System-Resource-Monitor.py
```

## Expected Output

- Files are renamed/moved/sorted based on script logic
- CPU/memory metrics printed in terminal for monitoring script
