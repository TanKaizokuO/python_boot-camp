# Basic Data Science Projects

## Topic

Starter data-science exercises combining notebook and script-based practice.

## What You Will Learn

- Basic data loading and transformation workflows
- Numerical operations with NumPy
- Tabular analysis with pandas

## What You Will Build

- Day-by-day data analysis exercises
- Small data processing scripts

## How to Run

Notebook:

```bash
jupyter notebook Basic-Data-Science-Projects/day_01.ipynb
```

Scripts:

```bash
python Basic-Data-Science-Projects/day_02.py
python Basic-Data-Science-Projects/day_03.py
python Basic-Data-Science-Projects/day_04.py
python Basic-Data-Science-Projects/day_05.py
```

## Expected Output

- Notebook visual/printed analysis results
- Script output showing data transformations and summaries
