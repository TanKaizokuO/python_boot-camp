# Basic Projects

## Topic

Mini command-line Python projects for practicing logic, user input, and script structure.

## What You Will Learn

- Taking user input and validating it
- String manipulation and control flow
- Building small utilities with reusable functions

## What You Will Build

- Caesar cipher tool
- Password strength checker
- Terminal task manager

## How to Run

From the project root:

```bash
python Basic_Projects/ceaser-cipher.py
python Basic_Projects/password-checkerr.py
python Basic_Projects/Terminal_Task_Manager.py
```

## Expected Output

- Interactive terminal prompts
- Processed text/password feedback
- Task manager actions printed in terminal
