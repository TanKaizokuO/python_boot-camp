# Data Handling Projects

## Topic

Practical scripts for manipulating CSV, JSON, and Excel data plus local productivity tools.

## What You Will Learn

- Reading/writing structured files
- Converting between common data formats
- Building utility-style CLI scripts

## What You Will Build

- CSV to JSON converter
- JSON flattener and JSON to Excel converter
- Contact book and tracker-style utilities

## How to Run

From the project root:

```bash
python Data-Handling-Project/CSV-TO-JSON.py
python Data-Handling-Project/JSON-Flattener.py
python Data-Handling-Project/JSON-to-Excel.py
python Data-Handling-Project/CLI_Contact_Book.py
python Data-Handling-Project/PersonalMovieTracker.py
```

## Expected Output

- Converted output files (JSON/Excel) where applicable
- Terminal-based menus or summaries for CLI utilities
