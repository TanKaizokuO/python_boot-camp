# 📋 Course Syllabus — Free Python Course

A complete, self-paced curriculum from zero to advanced Python.

---

## 🗺️ Learning Path

```
Phase 1 → Core Python (Notebooks)
Phase 2 → Projects (Apply what you learned)
Phase 3 → Advanced Python (Concurrency & Async)
Phase 4 → Real-World Skills (Web Scraping & Automation)
```

---

## Phase 1 — Core Python

Work through these notebooks in order. Each builds on the last.

| # | Module | File | Topics |
|---|--------|------|--------|
| 00 | Getting Started | [`00_intro/getting_started.ipynb`](./00_intro/getting_started.ipynb) | What is Python, how to run code, first program |
| 01 | Python Basics | [`1.basics.ipynb`](./1.basics.ipynb) | Variables, data types, operators, input/output, conditionals, loops |
| 02 | Intermediate Python | [`2.intermediate.ipynb`](./2.intermediate.ipynb) | Functions, scope, args/kwargs, lambda, map/filter |
| 03 | Comprehensions | [`3.comprehension.ipynb`](./3.comprehension.ipynb) | List, dict, set comprehensions; nested comprehensions |
| 04 | Generators | [`4.generaors.ipynb`](./4.generaors.ipynb) | `yield`, generator functions, lazy evaluation, `itertools` |
| 05 | Decorators | [`5.Decorators.ipynb`](./5.Decorators.ipynb) | Function wrappers, `@decorator` syntax, practical use cases |
| 06 | Errors & Logging | [`6.ErrorNLog.ipynb`](./6.ErrorNLog.ipynb) | `try/except/finally`, custom exceptions, `logging` module |
| 07 | OOP Basics | [`7.OOPs1.ipynb`](./7.OOPs1.ipynb) | Classes, objects, `__init__`, instance vs class variables, methods |
| 08 | Inheritance | [`8.Inheritance.ipynb`](./8.Inheritance.ipynb) | Single/multi inheritance, `super()`, method resolution order |
| 09 | OOP Advanced | [`9.OOPs2.ipynb`](./9.OOPs2.ipynb) | Dunder methods, properties, abstract classes, polymorphism |
| 10 | Pydantic | [`10. PYDANTIC.ipynb`](./10.%20PYDANTIC.ipynb) | Data validation, models, type hints, settings management |
