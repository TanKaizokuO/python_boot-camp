# Syllabus - Free Python Course

This syllabus maps every lesson and project in the repository into a clear self-paced sequence.

## Phase 1 - Core Python (Modules 01-10)

Recommended order: complete one module per day or 2-3 modules per week.

### Module 00 - Intro and Orientation

- Lesson file: [00_intro/getting_started.ipynb](./00_intro/getting_started.ipynb)
- Goal: understand how Python code runs and how to work through this course.

### Module 01 - Python Basics

- Lesson file: [1.basics.ipynb](./1.basics.ipynb)
- Topics: variables, data types, operators, conditions, loops, input/output.
- Outcome: write your first complete beginner scripts.

### Module 02 - Intermediate Concepts

- Lesson file: [2.intermediate.ipynb](./2.intermediate.ipynb)
- Topics: functions, parameters, return values, scope, lambda, map/filter.
- Outcome: break logic into reusable functions.

### Module 03 - Comprehensions

- Lesson file: [3.comprehension.ipynb](./3.comprehension.ipynb)
- Topics: list, dict, and set comprehensions; nested patterns.
- Outcome: write concise transformation code.

### Module 04 - Generators

- Lesson file: [4.generaors.ipynb](./4.generaors.ipynb)
- Topics: `yield`, lazy iteration, generator expressions.
- Outcome: process large data streams efficiently.

### Module 05 - Decorators

- Lesson file: [5.Decorators.ipynb](./5.Decorators.ipynb)
- Topics: higher-order functions, wrappers, decorator syntax.
- Outcome: add reusable cross-cutting behavior (logging, timing, validation).

### Module 06 - Errors and Logging

- Lesson file: [6.ErrorNLog.ipynb](./6.ErrorNLog.ipynb)
- Topics: exception handling, custom exceptions, logging patterns.
- Outcome: write safer code with meaningful debug output.

### Module 07 - OOP Foundations

- Lesson file: [7.OOPs1.ipynb](./7.OOPs1.ipynb)
- Topics: classes, objects, attributes, methods, constructors.
- Outcome: model real entities using object-oriented design.

### Module 08 - Inheritance

- Lesson file: [8.Inheritance.ipynb](./8.Inheritance.ipynb)
- Topics: base/child classes, method overriding, `super()`.
- Outcome: build extensible class hierarchies.

### Module 09 - OOP Advanced

- Lesson file: [9.OOPs2.ipynb](./9.OOPs2.ipynb)
- Topics: polymorphism, special methods, properties.
- Outcome: design cleaner, more expressive OOP APIs.

### Module 10 - Pydantic

- Lesson file: [10. PYDANTIC.ipynb](./10.%20PYDANTIC.ipynb)
- Topics: data validation, typed models, schema-driven coding.
- Outcome: validate structured data confidently.
