# Threading and Concurrency Module

## Topic

Threading and multiprocessing practice files for understanding parallel execution in Python.

## What You Will Learn

- Thread lifecycle and synchronization
- Multiprocessing and data sharing
- GIL effects on CPU-bound vs I/O-bound tasks

## What You Will Build

- Thread and process execution demos
- Lock/queue/value sharing examples
- Performance comparison experiments

## How to Run

From the project root:

```bash
python Thread_n_Concurrency/12_threads_concurrency/01_threading.py
python Thread_n_Concurrency/12_threads_concurrency/02_multiprocessing.py
python Thread_n_Concurrency/12_threads_concurrency/08_thread_lock.py
python Thread_n_Concurrency/12_threads_concurrency/11_process_queue.py
```

## Expected Output

- Interleaved thread/process logs
- Demonstrations of locking, queue communication, and process-safe values
