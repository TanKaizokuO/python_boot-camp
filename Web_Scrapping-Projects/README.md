# Web Scraping Projects

## Topic

Scripts for collecting and transforming web data using HTTP requests and HTML parsing.

## What You Will Learn

- Sending requests to websites and APIs
- Parsing HTML with BeautifulSoup
- Extracting and formatting useful information

## What You Will Build

- Crypto price trackers
- Hacker News post scraper
- Book scraper and wiki heading scraper
- Quote image utility

## How to Run

From the project root:

```bash
python Web_Scrapping-Projects/Crypto_Price_Tracker-1.py
python Web_Scrapping-Projects/Hacker-News_Top_Posts_Scraper.py
python Web_Scrapping-Projects/Scrape-Books.py
python Web_Scrapping-Projects/Scrap_wiki_Headings.py
```

## Expected Output

- Scraped text/data printed to terminal or saved by script
- Depending on script, generated files such as downloaded assets/images
